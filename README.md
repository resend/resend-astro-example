This example shows how to use Resend with Astro Actions.

## Instructions

1. Define environment variables in `.env` file.

```sh
cp .env.example .env
```

2. Install dependencies:

```sh
npm install
# or
yarn
  ```

3. Execute the following command:

```sh
npm run dev
```
